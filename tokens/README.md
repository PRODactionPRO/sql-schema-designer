# Token Studio Sync Directory

This directory is reserved for design tokens synchronized from Token Studio via Git.

Use this path in Token Studio:

- `Token storage location (file/folder path)`: `tokens`

Notes:

- Single-file sync example: `tokens/tokens.json`
- Multi-file/themes sync: `tokens` (folder mode)
- Current token sets in `tokens/tokens.json`:
  - `core` (primitive colors + radius)
  - `semanticLight`
  - `semanticDark`
  - `semanticEditor`
  - `typography` (font families, weights, sizes, text styles)
